# Box Shadow

Box Shadow is used to set the size of an element's drop shadow

## Box Shadow Sizes

The box-shadow utility has 6 settings (sizes) to choose from: 'none', 1, 2, 3, 4, and 5. Each setting changes the _size_ of the element's drop shadow. NCIDS has increased the opacity of the shadow from USWDS's 10% to the current 30% alpha value.

<br />

| Shadow HTML                                    | CSS Value                            | Mixin                       |
| :--------------------------------------------- | :----------------------------------- | :-------------------------- |
| <div className="shadow-none">Shadow None</div> | `'none'`                             | `@include u-shadow('none')` |
| <div className="shadow-1">Shadow 1</div>       | `'0 1px 4px 0 rgba(0, 0, 0, 0.3)'`   | `@include u-shadow(1)`      |
| <div className="shadow-2">Shadow 2</div>       | `'0 4px 8px 0 rgba(0, 0, 0, 0.3)'`   | `@include u-shadow(2)`      |
| <div className="shadow-3">Shadow 3</div>       | `'0 8px 16px 0 rgba(0, 0, 0, 0.3)'`  | `@include u-shadow(3)`      |
| <div className="shadow-4">Shadow 4</div>       | `'0 12px 24px 0 rgba(0, 0, 0, 0.3)'` | `@include u-shadow(4)`      |
| <div className="shadow-5">Shadow 5</div>       | `'0 16px 32px 0 rgba(0, 0, 0, 0.3)'` | `@include u-shadow(5)`      |
