The appearance of an item can be set by the following tokens and utilities.

<a href="/foundations/appearance/border" className="header-styled-link">
	Border
</a>

Set width, color, style, and radius of an item's borders.

<a href="/foundations/appearance/outline" className="header-styled-link">
	Outline
</a>

Set width and color of an item's outline.

<a href="/foundations/appearance/shadow" className="header-styled-link">
	Shadow
</a>

Set size of an item's drop shadow.

<a href="/foundations/appearance/opacity" className="header-styled-link">
	Opacity
</a>

Opacity tokens set the opacity (or transparency) of an item in increments of 10%.
