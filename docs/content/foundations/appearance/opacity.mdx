# Opacity

Opacity tokens set the opacity (or transparency) of an item in increments of 10%.

## Opacity in the U.S. Web Design System (USWDS)

Opacity utilities have not been modified from USWDS. Refer to [USWDS Opacity Tokens](https://designsystem.digital.gov/design-tokens/opacity/).

## For Designers: How to apply Opacity in the NCIDS Design Kit in Figma

Refer to the [Figma Guide](../../get-started/figma-guide) for how to apply Opacity to an item in Figma.
