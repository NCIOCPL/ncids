Change the spacing around an item and inside an item.

## Margin and Padding in USWDS

Margin and Padding utilities have not been modified from the U.S. Web Design System (USWDS). Refer to [USWDS Margin and Padding Utilities](https://designsystem.digital.gov/utilities/margin-and-padding/).

## Designer Notes

Showing the margin and padding around an item or inside an item can be done in your designs by utilizing spacing tokens in Figma. To learn more on how to apply these, visit the Spacing Units section of the [Figma Guide](https://designsystem-dev.cancer.gov/develop/get-started/figma-guide).
