# Heading

A customized display and label heading as well as h1-h6 styles utilizing USWDS theme tokens.

[View in Figma](https://www.figma.com/community/file/1288512334920909770/ncids-design-kit)

## Usage

- Use the display variant if you'd like the heading to be responsive between tablet and large tablet.
- Use the label variant for a horizontal rule under the section that extends the width of the page.

### Where to use

- Before a section of content.

### When to use

- When introducing a content section.

## Best Practices

- Title shouldn't exceed a specified character limit.
