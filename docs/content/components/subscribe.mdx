## Guidance

### When to use subscribe form

- This component should be used when including a way for users to subscribe to your content in the footer of your website. NCI uses GovDelivery (Granicus) as its main subscription service, and should pass the email address entered in this component through to GovDelivery's subscription page.
- This consists of a text input, and a button component from the design system.

### Usability guidance

- Validation messages or stylings should only appear after a user has interacted with a particular field or button within this component.
