# Guide Card with Image and Description Row (2 Cards)

The guide card with image and description row displays our custom Guide Card with Image and Description component in a grid row of 2 cards in which the cards height is equal across the row and the cards align with each other under their titles.

## Guide Card Row guidance

### When to use Guide Card with Image & Description Row variation

- When there is a need to for two specific audiences or groups of content

### When to consider something else

- If you don't have multiple audiences, use the Feature Card variation

### Additional considerations

- Minimum of two buttons and a maximum of 6 buttons for each variation

### Accessibility

- **Aria-Labels provide an accessible name for an element.** If there is no visible name for the element you can reference, use aria-label to provide the user with a recognizable accessible name.
- The aria-label of this component is related to the anchor element of the card (the aria-label is on the anchor tag)
