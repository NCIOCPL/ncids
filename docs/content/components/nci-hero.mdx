# Hero Image w/Button & Text Box

## When to use Hero Image w/Button & Text Box variation

- To introduce the page visually and catch the user's attention
- To highlight a main call-to-action

## When to consider something else

- When you need to display groups of information or long chunks of text

## Variations

- Light text box variation is meant for darker images or illustrations
- Dark text box variation is meant for lighter images or illustrations
