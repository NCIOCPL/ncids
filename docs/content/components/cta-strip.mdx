<br />

## CTA Strip guidance

### When to use the CTA Strip only variation

- When you want to have multiple call-to-actions for an informational landing page
- When a hero image would deter the user from quickly finding the information they need

### When to consider something else

- When you want to have a hero image and a tagline
- When you need to display groups of information or long chunks of text
