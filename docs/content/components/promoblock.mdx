# Promo Block with Image / Alternating Promo Block with Image

- Promoblock highlights one specific piece of content.

When to use Alternating Promo Block with Image

- Used for promotional content to highlight a single subject with left-anchored or right-anchored visual
- Used when highlighting content with multiple Promoblocks

When to consider something else

- If multiple links are needed, look at guide card component variations

Variations

- Light text box variation is meant for darker images or illustrations
- Dark text box variation is meant for lighter images or illustrations
  <br />

Additional considerations

- Image on the left should always be the first card unless there is an alternating guide card above

# Promo Block w/o Image (Default Promoblock)

When to use Promo Block w/o Image

- Used for promotional content to highlight a single subject without a visual image

When to consider something else

- If a visual image is needed to accompany promotional content, use the Alternating Promo block variation

Variations

- Light text box variation is meant for darker images or illustrations
- Dark text box variation is meant for lighter images or illustrations
